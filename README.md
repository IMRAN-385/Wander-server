# 🌍 Wanderlust Server

Backend API for the Wanderlust Travel Discovery Platform built with **Node.js, Express.js, TypeScript, MongoDB, and JWT Authentication**.

## 🚀 Live API

https://wander-server.onrender.com

## 🌐 Frontend

https://wander-client-gilt.vercel.app/

## ✨ Features

- JWT Authentication
- Password Hashing (bcrypt)
- MongoDB Database
- Mongoose
- REST API
- Protected Routes
- CRUD Operations
- Error Handling
- CORS Configuration
- Environment Variables

## 🛠 Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT
- bcrypt
- Cookie Parser
- CORS
- Dotenv

## ⚙️ Environment Variables

### Local

```env
PORT=5000

MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

CLIENT_URL=http://localhost:3000

NODE_ENV=development
```

### Production (Render)

```env
MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

CLIENT_URL=https://wander-client-gilt.vercel.app

NODE_ENV=production
```

## Installation

```bash
git clone https://github.com/IMRAN-385/Wander-server.git

cd Wander-server

npm install

npm run dev
```
